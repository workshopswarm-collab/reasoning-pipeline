# Baseball Metrics Sources

## Preferred neutral baseline sources

- **Baseball Reference** — deep historical and current player/team/season context
- **MLB Standings** — official current standings and race context
- **MLB Transactions** — official roster and injured-list movement
- **MLB News** — official injury, storyline, prospect, and team-state context

## Usage guidance

- Use Baseball Reference for baselines, player histories, team pages, standings context, and league leaders.
- Use MLB Standings for official current playoff/division context.
- Use MLB Transactions for live roster and injured-list changes.
- Use MLB News for narrative, prospect, and injury context.
- Do not evaluate a baseball team from standings alone; combine team strength, player quality, injuries, transactions, and recent form.
