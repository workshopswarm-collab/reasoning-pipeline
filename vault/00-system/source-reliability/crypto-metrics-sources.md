# Crypto Metrics Sources

## Preferred neutral baseline sources

- **CoinGecko** — broad price, market cap, FDV, volume, exchange-aggregated data
- **Protocol / foundation docs** — best for first-party positioning, not neutral valuation
- **Legal / policy overviews** — useful for regulation framing, not price discovery by themselves

## Usage guidance

- Use CoinGecko for broad cross-asset comparisons and historical market context.
- Use first-party docs to understand how an ecosystem describes itself.
- Never rely on one source alone for crypto evaluation; combine market data, protocol context, and regulatory framing.
