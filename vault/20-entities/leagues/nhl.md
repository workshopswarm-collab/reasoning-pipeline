# Summary

The NHL is the core hockey league entity because low-scoring variance, goaltending, star skater value, and playoff dynamics all derive structure through it.

## What this entity is

The premier professional ice hockey league in North America and the main reference context for hockey markets.

## Why it matters

The NHL matters because it structures team quality, star-player value, postseason randomness, and public attention for top-level hockey analysis.

## Current state

Research should focus on playoff structure, goaltending, roster depth, injuries, and whether star-driven edge overcomes low-scoring variance.

## Key strengths

- strong league identity and clear competitive framework
- rich team/player context for structured analysis

## Key weaknesses

- variance can obscure underlying team quality more than in many other major sports

## Important recent changes

- remains the main reference framework for pro hockey evaluation

## Open questions

- how much persistent contender edge exists in a high-variance postseason environment
