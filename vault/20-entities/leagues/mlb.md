# Summary

MLB is the central baseball league entity because team evaluation, player availability, standings, and playoff structure all inherit from league design and season length.

## What this entity is

The premier professional baseball league spanning the American and National Leagues.

## Why it matters

MLB context is necessary for interpreting division races, long-season variance, roster management, and postseason access.

## Current state

Research should track standings, transactions, injuries, seasonality, and league-wide competitive context.

## Key strengths

- deep historical data and team/player structure
- strong regular-season and playoff context for futures-style analysis

## Key weaknesses

- long seasons can create noisy short-run evaluation if context is ignored

## Important recent changes

- remains the core reference framework for baseball market analysis

## Open questions

- how much postseason structure and division context distort underlying team quality signals
