# Crypto Onchain & Fundamentals Sources

## Preferred sources

- **CoinGecko** — broad market cap, price, volume, and market-structure context
- **Etherscan** — direct Ethereum chain inspection and transaction validation
- **L2BEAT** — Ethereum L2 ecosystem structure and security assumptions
- **Token Terminal** — fees, revenue, TVL, and cross-project fundamentals

## Usage guidance

- Use CoinGecko for broad market comparison and price/market-cap context.
- Use Etherscan when you need direct evidence from Ethereum transactions, contracts, or addresses.
- Use L2BEAT to reason about Ethereum scaling ecosystems and L2 risk distinctions.
- Use Token Terminal when comparing economic activity and protocol fundamentals.
- Prefer combining market data, onchain evidence, and architecture/risk context rather than relying on price narratives alone.
