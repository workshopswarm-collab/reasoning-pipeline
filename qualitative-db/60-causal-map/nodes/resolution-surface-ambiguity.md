# Resolution surface ambiguity

## What this node means

- Uncertainty remains about the exact governing proof surface, benchmark print, or qualifying event representation for settlement.

## When it applies

- Use when the analyst understands the likely mechanism but has not yet captured decisive proof from the governing surface.
- This node often appears when a contract is venue-specific or publication-specific.

## Boundaries

- This is not the same as the event being unlikely or not occurring.
- If the governing surface has already been directly captured, this node should probably be inactive.
