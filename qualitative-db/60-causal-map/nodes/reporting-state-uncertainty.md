# Reporting-state uncertainty

## What this node means

- Uncertainty remains because the decisive reporting or publication state has not yet updated, finalized, or propagated clearly.

## When it applies

- Use when the likely real-world state may be known soon, but the official reporting surface is still lagged or incomplete.

## Boundaries

- This node is about reporting state, not necessarily about the underlying event probability.
